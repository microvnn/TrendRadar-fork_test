# Tin tức chứa các từ này sẽ tự động bị loại trừ và không xuất hiện trong thông báo đẩy.
#
# Cách sử dụng:
#   sốc               Viết trực tiếp từ, tin tức chứa "sốc" sẽ bị lọc
#   /cờ bạc|cá độ/    Dùng /.../ để khớp nhiều từ (phân cách bằng |)

# ───────────────────────────────────────────────────────────────
#                        Khu vực định nghĩa nhóm từ
# ───────────────────────────────────────────────────────────────
# Viết các từ khóa bạn muốn theo dõi ở đây.
# Các nhóm từ được phân cách bằng dòng trống, các từ khóa trong cùng một nhóm có quan hệ "HOẶC".
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      Tổng quan cú pháp (Tham khảo nhanh)     │
# └─────────────────────────────────────────────────────────────┘
#
# Cú pháp từ khóa:
#   từ_khóa           Từ khóa thông thường, khớp nếu tiêu đề chứa từ này
#   /regex/           Khớp biểu thức chính quy (tự động bỏ qua chữ hoa/thường)
#   từ_khóa => alias  Chỉ định tên hiển thị (alias) cho từ khóa
#   [tên_nhóm]        Dòng đầu tiên của nhóm từ, chỉ định tên hiển thị cho cả nhóm
#   +từ_khóa          Từ bắt buộc, tất cả các từ bắt buộc phải khớp mới được tính là khớp
#   !từ_khóa          Từ loại trừ, nếu khớp sẽ loại bỏ tin tức đó (chỉ trong nhóm từ hiện tại)
#   @số               Giới hạn nhóm từ này hiển thị tối đa bao nhiêu tin
#
# Thứ tự ưu tiên tên hiển thị:
#   1. Có tên nhóm -> Hiển thị tên nhóm
#   2. Không có tên nhóm -> Hiển thị các alias nối với nhau (dùng " / " để nối)
#   3. Không có alias -> Hiển thị chính từ khóa đó
#
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      Cách dùng cơ bản (Khuyên dùng cho người mới) │
# └─────────────────────────────────────────────────────────────┘
#
# 1. Đơn giản nhất: Viết trực tiếp từ khóa
#    ────────────────────
#    Huawei
#
#    Hiệu quả: Khớp tất cả tin tức chứa "Huawei"
#
#
# 2. Gom nhiều từ khóa vào một nhóm
#    ────────────────────
#    Huawei
#    HarmonyOS
#    Nhậm Chính Phi
#
#    Hiệu quả: Khớp tin tức chứa "Huawei" hoặc "HarmonyOS" hoặc "Nhậm Chính Phi", hiển thị chung là "Huawei / HarmonyOS / Nhậm Chính Phi"
#
#
# 3. Đặt tên cho nhóm từ (Khuyên dùng)
#    ────────────────────
#    [Huawei]
#    Huawei
#    HarmonyOS
#    Nhậm Chính Phi
#
#    Hiệu quả: Như trên, nhưng tên hiển thị là "Huawei" (gọn gàng hơn)
#
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      Cách dùng nâng cao (Tùy chọn)            │
# └─────────────────────────────────────────────────────────────┘
#
# 4. Dùng biểu thức chính quy để khớp nhiều từ (trong 1 dòng)
#    ────────────────────
#    /Huawei|HarmonyOS|Nhậm Chính Phi/ => Huawei
#
#    Hiệu quả: Khớp tin tức chứa "Huawei" hoặc "HarmonyOS" hoặc "Nhậm Chính Phi", hiển thị là "Huawei"
#    Giải thích: Trong /.../ dùng | để phân cách nhiều từ, phía sau => là tên hiển thị
#
#    💡 Không hiểu regex? Hỏi AI:
#       "Giúp tôi viết một biểu thức chính quy, khớp văn bản chứa 'Huawei' hoặc 'HarmonyOS' hoặc 'Nhậm Chính Phi',
#        yêu cầu định dạng: /regex/ => tên hiển thị"
#
#
# 5. Khớp chính xác từ tiếng Anh (tránh khớp sai)
#    ────────────────────
#    /\bAI\b/i => AI
#
#    Giải thích: \b biểu thị ranh giới từ, tránh khớp "AI" trong "MAIL"
#          /i biểu thị bỏ qua chữ hoa/thường, "ai", "AI", "Ai" đều khớp
#
#    💡 Không hiểu regex? Hỏi AI:
#       "Giúp tôi viết một biểu thức chính quy, khớp chính xác từ tiếng Anh 'AI' (không khớp AI trong MAIL),
#        bỏ qua chữ hoa/thường, yêu cầu định dạng: /regex/i => tên hiển thị"
#
#
# 6. Loại trừ nội dung cụ thể
#    ────────────────────
#    [Công ty Apple]
#    Apple
#    !Trái cây
#    !Vườn cây
#
#    Hiệu quả: Khớp "Apple" nhưng loại trừ tin tức chứa "Trái cây" hoặc "Vườn cây"
#    Giải thích: Từ bắt đầu bằng ! biểu thị "loại trừ"
#
#
# 7. Giới hạn số lượng hiển thị
#    ────────────────────
#    [Tin công nghệ]
#    Công nghệ
#    @5
#
#    Hiệu quả: Hiển thị tối đa 5 tin tức khớp
#    Giải thích: @số biểu thị giới hạn số lượng
#
#
# 8. Bắt buộc phải chứa nhiều từ đồng thời
#    ────────────────────
#    +Apple
#    +Sự kiện ra mắt
#
#    Hiệu quả: Phải chứa đồng thời "Apple" và "Sự kiện ra mắt" mới khớp
#    Giải thích: Từ bắt đầu bằng + biểu thị "bắt buộc chứa"
#
# ───────────────────────────────────────────────────────────────
